재료창고 사용법
기본

이름 입력 → 수량/메모(선택) → 카테고리 선택 → + 추가 또는 엔터

수정

이름이나 수량을 클릭하면 바로 편집 가능. 엔터로 저장, Esc로 취소.

카테고리 이동

항목 왼쪽 ⠿ 핸들을 잡고 다른 카테고리 헤더에 드롭.

냉동 표시

항목에 마우스를 올리면 ❄️ 버튼이 나타남. 클릭하면 냉동 상태로 표시되고 AI 프롬프트에 자동 반영.

AI 요리 추천

오른쪽 패널에서 프롬프트 생성 → 복사 → AI 챗봇에 붙여넣기.

백업

데이터는 브라우저 로컬에 저장됨. 다른 기기로 옮기거나 백업하려면 백업 내보내기 → JSON 파일 저장 → 다른 곳에서 백업 불러오기.
